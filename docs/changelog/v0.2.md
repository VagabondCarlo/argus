# v0.2 — Multi-Asset Broadcast + Two-Tier Channel System

**Status:** Released

## What Was Built

- **4-asset-class coverage**: forex (9 major pairs), precious metals (5 futures), crypto (9 assets) added via yfinance extended universe
- **Multi-asset analyzer** (`analyst/sentiment/analyzer_extended.py`): three-committee framework adapted for non-equity assets — currency policy divergence, inflation regime for metals, institutional flow for crypto
- **3× daily broadcasts**: 8:15 AM pre-market, 12:00 PM midday, 4:30 PM next-day preview — runs 7 days/week
- **Two-tier Telegram channel structure**:
  - Tier 1 (free public): 2nd-best pick per asset class with one-line execution hint
  - Tier 2 (paid private): top 3 per class with full entry, stop-loss, price target, and committee reasoning
- **Execution suggestion engine** (`analyst/signals/execution.py`): conservative / moderate / aggressive approach per signal per asset class, always framed as suggestions with DYOR disclaimer
- **Suggestion-only language**: all guidance framed as "some traders consider..." — not directives
